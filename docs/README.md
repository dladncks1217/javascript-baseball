# 프리코스 2주차 - 기능 구현 목록

## 🎳 게임 진행 관련 기능

- [x] **게임 시작 기능**

  - [x] 게임 시작 문구 출력
  - [x] 정답 제작 함수 호출을 통해 정답 생성

- [x] **정답 맞추기 기능**

  - [x] 유저 입력값 유효 여부 확인

  - [x] 정답 맞추면 재시작 여부 문구 출력
  - [x] 입력값에 따른 결과 처리
  - [x] 1 또는 2가 아니라면 Error Throw
